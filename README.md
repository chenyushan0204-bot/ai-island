<h1 align="center">🏝️ AI Island</h1>
<p align="center">一个住在 Mac 状态栏里的小家伙，陪你写代码的桌面搭子。</p>

---

## ✨ 不切终端也能干活

权限弹窗直接桌面上点 Allow / Deny，不用切回黑乎乎的终端。再接上 **Telegram 远程审批**，人不在电脑前也能 yes yes yes。

## 📊 一眼看懂所有窗口

Dashboard + Session HUD，几个窗口在跑、各自在干嘛（思考？打字？卡住了？）统统挂在桌宠下面，瞄一眼就知道，不用一个个切过去猜。

## 🎭 情绪价值拉满

- 任务完成蹦起来 🎉
- 报错了抖一抖 😤
- 卡壳了歪头思考 🤔
- 眼睛会跟着你的鼠标转
- 闲了还会打哈欠
- 嫌碍眼？缩成一个小不点躲到屏幕边上

## 🤝 多 Agent 好脾气

Claude Code、Codex 都接得住，各管各的会话，互不打架。

## 🌏 语言随便切

英文、简中、繁中、日本語、한국어 — 菜单气泡全适配，一句话的事。

## 🏠 远程也管得着

Remote SSH 一键隧道，远端 AI 在跑，本地桌宠照样蹦跶给你看。

## 🎨 形象不喜欢？换！

内置主题系统，想换什么形象都行，扒拉一下设置就能换。

---

## 📦 怎么装

开源 **AGPL-3.0**，随便 fork 随便改。

不想折腾的直接下 DMG：

| 架构 | 下载 |
|------|------|
| Apple Silicon (M1/M2/M3/M4) | [AI-Island-0.1.0-arm64.dmg](https://github.com/chenyushan0204-bot/ai-island/releases/latest) |
| Intel Mac | [AI-Island-0.1.0-x64.dmg](https://github.com/chenyushan0204-bot/ai-island/releases/latest) |

> ⚠️ 还没公证，首次打开请在 **系统设置 → 隐私与安全性** → 点「仍要打开」

想自己改代码的：

```bash
git clone https://github.com/chenyushan0204-bot/ai-island.git
cd ai-island
npm install
npm start
```
